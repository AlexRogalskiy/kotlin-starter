Kotlin Build Script Compilation

Script compilation happens in 4 steps:

  1. Extract buildscript block, compile it against parent project classpath (up to buildSrc), evaluate it against current project.
  2. Extract plugins block, compile it against the same classpath as the previous step, evaluate it against current project.
  3. MAGIC (generate Kotlin code based on information contributed by previous steps).
  4. Compile whole script against classpath contributed by previous steps.
